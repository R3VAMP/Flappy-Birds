# Flappy-Birds

Made the Flappy Bird Game in Python using pygame.

## Requirements
    - Python 3
    - Pygame

## How to install package
Open the terminal and type pip to check whether pip is installed or not
Once installed type the below command in terminal
- pip install pygame
